# Bootstarter WordPress theme

Blank WordPress Theme for new projects based on Bootstrap (3.3.7) enhanced with [Bootstarter template](https://github.com/alexweblab/bootstarter).

## Plugins:
* jQuery (3.2.1)
* Select2 (4.0.4)
* Jasny Bootstrap (3.1.3)
* Flickity (2.0.10)
* Fancybox (3.1.28)
* CSS User Agent
